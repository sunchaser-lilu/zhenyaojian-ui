## 使用

### clone

```bash
~$ git clone https://github.com/llnancy/zhenyaojian-ui.git
```

### yarn

```bash
~$ yarn install
~$ yarn serve
```

### or npm

```bash
~$ npm install
~$ npm run serve
```
